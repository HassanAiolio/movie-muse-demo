# MovieMuse Frontend

  ## Setup

  1. Install dependencies:
     ```
     npm install
     ```

  2. Create a `.env` file (copy from `.env.example`):
     ```
     VITE_BACKEND_URL=https://your-backend-url.com
     ```

  3. Run the dev server:
     ```
     npm run dev
     ```

  4. Build for production:
     ```
     npm run build
     ```
